rabbitmq

helm repo add stable https://charts.helm.sh/stable

helm repo update

helm install mu-rabbit stable/rabbitmq

to acess rabbitmq ui: 

kubectl port-forward svc/mu-rabbit-rabbitmq 15672:15672

to get credentials for rabbimq ui:username:user && password :

kubectl get secret  mu-rabbit-rabbitmq -o jsonpath="{.data.rabbitmq-password}" | base64 --decode
